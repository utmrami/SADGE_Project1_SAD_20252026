# UTM Clinic Appointment System

A simple web-based appointment system designed to improve access to clinic services for students and staff at Universiti Teknologi Malaysia (UTM).

---

## Features

- Book clinic appointments online  
- View available doctors and time slots  
- Reschedule or cancel appointments  
- Receive reminders for upcoming visits  
- Access basic information about treatments  

---

## Why This Project?

The current clinic process requires users to physically visit the clinic to:
- Check doctor availability  
- Book appointments  
- Wait in long queues  

This project aims to make the process more convenient and reduce unnecessary waiting time.

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js / Firebase (TBD)  
- **Database:** Firebase / simple database solution
